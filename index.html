Werners Beispielseite

Verwenden von Sass und Gulp

Hier ist eine kurze Anleitung zur Einrichtung:

  • Git Repo klonen
  • Starten Sie, indem Sie "npm install" in der Befehlszeile ausführen.
  • Führen Sie "gulp" in der Befehlszeile aus.

Projektinfo

Dies ist ein Starter-Front-End-Projekt, das SCSS- und JS-Dateien mit Gulp 4 kompiliert.
